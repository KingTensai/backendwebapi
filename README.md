# backendwebapi
Volg deze stappen om het project op je lokale machine uit te voeren:

1.  **Project clonen:**
    git clone https://github.com/KingTensai/backendwebapi
    cd <project-map>

2.  **Configuratie:**
     env. file in de correcte directory zetten 
 
3.  **Server starten:**
   node index.js


## Bronvermeldingen
